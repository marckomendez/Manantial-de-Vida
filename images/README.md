# Directorio de Imágenes

Este directorio está destinado para almacenar las imágenes utilizadas en el sitio web.

## Tipos de Imágenes Recomendadas

- **Logo de la iglesia**: logo.png (tamaño recomendado: 200x200px)
- **Imagen principal (hero)**: hero-image.jpg (tamaño recomendado: 1920x1080px)
- **Fotos de ministerios**: ministry-*.jpg (tamaño recomendado: 800x600px)
- **Fotos de eventos**: event-*.jpg (tamaño recomendado: 800x600px)
- **Galería**: gallery-*.jpg (tamaño recomendado: 1200x800px)

## Formato de Nombres

Se recomienda usar nombres descriptivos en minúsculas con guiones:
- `logo-iglesia.png`
- `hero-principal.jpg`
- `ministerio-jovenes.jpg`
- `evento-navidad-2024.jpg`

## Optimización

Para un mejor rendimiento del sitio, se recomienda:
- Comprimir las imágenes antes de subirlas
- Usar formatos web optimizados (WebP cuando sea posible)
- Mantener el tamaño de archivo por debajo de 500KB para imágenes grandes
- Usar herramientas como TinyPNG o ImageOptim para comprimir imágenes
